# 齐鲁工大信息系统导航(一号通)使用指南

## 一号通是什么

- 齐鲁工大的大部分服务现均已接入一号通，

只需要记住和输入一次凭证，就可以使用多种服务，

无需重复登录，极大地方便了广大师生使用校内各类服务

## 如何访问信息系统导航

- 通过网址访问

访问[信息系统导航网站](https://sso.qlu.edu.cn)，

通过统一身份认证系统登录后即可进入信息系统导航主页：

<img src="/img/infocenter-home.png" width=100% />

信息导航系统主页展示的是一些常用的热门服务，

如果在主页没有找到想要访问的服务，可以点击页面顶部的服务按钮，

转到服务页面可以找到几乎所有校内提供的服务

<img src="/img/infocenter-services.png" width=100% />
